
This folder contains files required for integration with Microsoft Sentinel.

- sentinel_template.json
    - JSON Template file for the Sandfly Results Custom Table

- sentinel_template_hosts.json
    - JSON Template file for the Sandfly Host Assets Custom Table

- sentinel_template_sshkey.json
    - JSON Template file for the Sandfly SSH Keys Custom Table
