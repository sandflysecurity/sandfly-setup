
This folder contains files required for integration with Elasticsearch/Kibana.

- sandfly_elastic_dashboards.ndjson
    - Dashboards and reports that can be imported into Elasticsearch/Kibana

- sandfly_elastic_mappings.json
    - Mappings file to configure the Sandfly Elasticsearch index type mappings

- sandfly_elastic_settings.json
    - Settings file to configure the Sandfly Elasticsearch index total_fields.limit value
